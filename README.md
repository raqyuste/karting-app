# karting-app
World Kart Championship
https://raqyuste.github.io/karting-app

## Installing
```sh
git clone git@github.com:raqyuste/karting-app.git
cd karting-app && npm install
```

## Local serve
```sh
npm start
```

## Building
```sh
npm run build
```

```sh
npm run build:dev
```

## Testing
```sh
npm test
```
